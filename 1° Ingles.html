Unidad 1: Tiempos Verbales en Inglés

A. Presente Simple

Uso: Describir sistemas y circuitos electrónicos que funcionan de manera habitual o general.

Estructura:

  • Afirmativa: Sujeto + verbo base (añadir 's' o 'es' en 3ª persona singular).
    Example: The circuit transmits data continuously.
  • Negativa: Sujeto + do/does + not + verbo base.
    Example: The device does not filter the signal.
  • Interrogativa: Do/Does + sujeto + verbo base?
    Example: Does the system operate at low voltage?

Ejemplos técnicos:

  • The processor executes instructions sequentially.
  • Electronic components generate heat during operation.

B. Condicional Cero (Zero Conditional)

Estructura: If + Present Simple, Present Simple

If clauses describen hechos generales, leyes o reglas.

Uso: Para describir reglas o hechos comprobados en sistemas electrónicos.

Ejemplo:

  • If the power goes off, the system shuts down immediately.
  • If the button is pressed, the LED lights up.

C. Pasado Simple

Verbo to be:

  • Afirmativo: I/He/She/It was; You/We/They were.
  • Negativo: I/He/She/It was not (wasn't); You/We/They were not (weren't).
  • Interrogativo: Was + sujeto? / Were + sujeto?

Verbos regulares e irregulares:

  • Regulares: verbo base + “-ed” (e.g., installed, tested).
  • Irregulares: formas específicas (e.g., ran, built, sent).

Indicadores temporales: yesterday, last week, two days ago, in 2019, etc.

Uso: Para narrar hechos terminados en el pasado, como la instalación de un equipo o la reparación de un dispositivo.

Ejemplo:

  • The technician installed the new hardware yesterday.
  • The circuit failed during the test last week.

D. Presente Perfecto

Uso: Narrar hechos recientes o con relevancia en el presente, indicar experiencias o resultados.

Estructura: have/has + participio pasado.

Estructuras:

  • Afirmativa: Subject + have/has + past participle
  • Negativa: Subject + have/has + not + past participle
  • Interrogativa: Have/Has + subject + past participle?

Ejemplos técnicos:

  • We have upgraded the firmware to the latest version.
  • The system has been repaired after the malfunction.

Participios usados como adjetivos: broken, installed, damaged.

Example: The broken cable needs replacement.

Unidad 2: Habilidades de Comunicación

A. Informes

Descripción y análisis de gráficos electrónicos (por ejemplo, gráficos de voltaje, consumo o rendimiento).

Uso de vocabulario técnico: increase, decrease, remain stable, peak, voltage, current, frequency, efficiency.

B. Narración

Uso del pasado simple para contar eventos técnicos o procesos realizados.

Uso de conectores de secuencia: first, then, after that, finally.

Ejemplo:

  • First, we tested the circuit.
  • Then, we recorded the voltage levels.
  • Finally, we analyzed the data.

C. Uso de preguntas

Preguntas con Why? para pedir explicaciones Técnicas:

  • Why did the system fail during the test?

Preguntas comparativas:

  • Who is faster at processing data, processor A or processor B?
  • Which is better, HDD or SSD for storage?

Unidad 3: Adjetivos y Comparaciones

A. Adjetivos técnicos

Referentes a software y hardware:

  • compatible, digital, mechanical, wireless
  • efficient, advanced, reliable, portable

B. Medidas y dimensiones

Cómo expresar o preguntar medidas:

  • width (ancho), length (largo), height (alto), depth (profundidad).

Ejemplo:

  • What's the height of the screen?
  • How wide is the motherboard?

C. Comparativos y superlativos

Adjetivos cortos:

  • Comparativo con -er: taller, faster, smaller
  • Superlativo con -est: tallest, fastest, smallest

Adjetivos largos:

  • Comparativo con more: more powerful, more efficient
  • Superlativo con most: most advanced, most reliable

Formas irregulares:

  • good → better → best
  • bad → worse → worst

Ejercicios Prácticos

Completa las oraciones

1. The system ______ (shut down) automatically when it overheats. (Present Simple)

2. If the temperature ______ (rise) too much, the fan ______ (activate). (Zero Conditional)

3. Yesterday we ______ (test) the new components. (Past Simple)

Traduce al inglés

1. El procesador ejecuta instrucciones rápidamente.

2. Si presionas este botón, el LED se enciende.

3. ¿Por qué falló el sistema durante la prueba?
