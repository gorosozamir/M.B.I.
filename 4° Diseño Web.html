1. Introducción al Diseño Web

Definición

El diseño web es la disciplina que abarca la planificación, diseño e implementación de sitios web, integrando elementos gráficos, estructurales y funcionales para crear interfaces digitales atractivas y usables.

Diferencia con el diseño convencional

Mientras el diseño convencional se enfoca en medios físicos o estáticos, el diseño web debe considerar la interactividad, navegabilidad y adaptabilidad a múltiples dispositivos y plataformas digitales.

Consideraciones clave

  • Navegabilidad: Facilitar el acceso y desplazamiento intuitivo por el sitio.
  • Interactividad: Incorporar elementos que respondan a acciones del usuario.
  • Usabilidad: Garantizar que el sitio sea fácil de usar y accesible.
  • Arquitectura de la información: Organizar y estructurar el contenido para su comprensión.
  • Integración de medios: Uso coordinado de audio, texto, imágenes, enlaces y video para enriquecer la experiencia.

El diseño web es una rama del diseño multimedia, ya que combina diferentes tipos de medios digitales para comunicar eficazmente.

2. Fundamentos tecnológicos

2.1 Sistemas distribuidos

Son sistemas donde los componentes ubicados en diferentes nodos de red cooperan para lograr un objetivo común, ofreciendo escalabilidad y tolerancia a fallos.

2.2 Paradigmas de programación

Programación estructurada

Basada en la división del programa en bloques o funciones, facilitando la claridad y mantenimiento.

Programación funcional

Se centra en el uso de funciones puras y evita estados mutables, mejorando la modularidad y paralelismo.

3. Hipertextos e Hipermedios

Definición

El hipertexto es un sistema de organización de información que permite la conexión no lineal mediante enlaces; el hipermedio extiende esta idea incorporando múltiples tipos de medios (texto, imagen, audio, video).

Aplicación

Base para la construcción de páginas web, facilitando la navegación a través de enlaces.

HTML es el estándar tecnológico que soporta estos conceptos, estructurando el contenido y sus enlaces en la web.

4. Herramientas y tecnologías

  • Adobe Dreamweaver: Editor visual y de código para crear y gestionar sitios web.
  • HTML5: Lenguaje de marcado estándar para estructurar contenido web, con soporte para multimedia y semántica avanzada.
  • CSS3: Lenguaje de estilos para definir la presentación visual de documentos HTML, incluyendo efectos y diseño responsivo.

5. HTML (HyperText Markup Language)

5.1 Introducción y estructura

HTML es el lenguaje fundamental para construir páginas web.

Un documento HTML se compone de una cabecera (<head>) que contiene metadatos y enlaces a recursos, y un cuerpo (<body>) donde se ubica el contenido visible.

5.2 Elementos HTML

  • Incluye etiquetas y atributos para estructurar texto, manipular fuentes, insertar vínculos a sitios, archivos, imágenes, videos y sonido.
  • Soporta listas ordenadas y no ordenadas, mapas de imagen, tablas con atributos para organizar datos, y formularios para interacción con el usuario.
  • La etiqueta <div> es un contenedor genérico para agrupar elementos y aplicar estilos o scripts.

6. CSS (Cascading Style Sheets)

  • CSS complementa a HTML definiendo estilos visuales.
  • Utiliza selectores y una sintaxis específica para aplicar reglas a elementos.
  • Permite agrupar múltiples declaraciones y heredar estilos, facilitando el mantenimiento y separación entre contenido y diseño.
  • Es fundamental para asegurar compatibilidad y diseño adecuado en distintos navegadores.

El uso de HTML estricto junto con CSS mejora la accesibilidad y consistencia visual.

7. Desarrollo y prueba de sitios web

El diseño y desarrollo de páginas web simples implica la creación de contenido estructurado y estilizado.

La prueba en distintos navegadores es esencial para garantizar un comportamiento consistente y detectar incompatibilidades.

8. Publicación y puesta en línea

  • Incluye el registro y delegación de dominios, gestión de servidores DNS.
  • Los sitios se suben al servidor mediante clientes FTP u otros métodos.
  • Se diferencian los sitios web tradicionales de los blogs, que suelen tener estructura dinámica y actualización frecuente.

9. Introducción a PHP

  • PHP es un lenguaje de programación del lado del servidor.
  • Permite declarar variables y constantes, manejar tipos de datos, y usar funciones básicas.
  • Soporta operadores aritméticos, relacionales y lógicos.
  • Implementa estructuras condicionales y de repetición.
  • Permite funciones definidas por el usuario con paso por valor o referencia.
  • Soporta vectores y matrices, y manipulación avanzada de cadenas.

10. Frameworks en desarrollo web

  • Son conjuntos de herramientas y librerías que facilitan el desarrollo web estructurado.
  • Ofrecen propiedades como modularidad, reutilización y soporte para programación orientada a objetos.
  • Su uso está extendido y en evolución constante.
  • Ejemplos incluyen frameworks para PHP, JavaScript y otros lenguajes.
