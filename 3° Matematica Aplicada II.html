1. Números Reales y Expresiones Algebraicas

Repaso general de los números reales

Los números reales incluyen todos los números racionales e irracionales, pudiendo representarse en la recta numérica. Comprenden números enteros, fracciones, decimales finitos e infinitos no periódicos.

Σ Propiedad importante: Los números reales cumplen con las propiedades de cuerpo: conmutativa, asociativa, distributiva, existencia de elemento neutro y existencia de inverso.

Propiedades de los números reales

  • Conmutativa: a + b = b + a ; a × b = b × a
  • Asociativa: (a + b) + c = a + (b + c)
  • Distributiva: a × (b + c) = a × b + a × c
  • Elemento neutro: a + 0 = a, a × 1 = a
  • Elemento opuesto y recíproco: Para cada a, existe -a; para cada a ≠ 0 existe 1/a

Escritura e interpretación de intervalos

Los intervalos expresan subconjuntos de la recta real.

  • Intervalo cerrado: [a, b] incluye a y b
  • Intervalo abierto: (a, b) excluye a y b
  • Intervalo semiabierto: [a, b) o (a, b] incluyen sólo uno de los extremos

Casos de factoreo

Descomposición en factores de expresiones algebraicas:

  • Factor común: a×b + a×c = a(b + c)
  • Trinomio cuadrado perfecto: a² + 2ab + b² = (a + b)²
  • Diferencia de cuadrados: a² - b² = (a - b)(a + b)

Operatoria y simplificación de expresiones algebraicas

Simplificación mediante suma, resta, multiplicación y división de términos semejantes. Uso de leyes de exponentes para consolidar potencias. Reducción de fracciones algebraicas y racionalización.

2. Funciones: Concepto, Representación y Aplicación

Concepto de función

Una función es una relación que asigna a cada elemento del conjunto dominio exactamente un elemento del conjunto codominio. Notación común: f(x), donde x es variable independiente.

Σ Notación: f: A → B, donde f(x) = y significa que la función f asigna al elemento x del dominio A el elemento y del codominio B.

Identificación de elementos y características de una función

  • Dominio: Conjunto de valores posibles para la variable independiente
  • Codominio: Conjunto al que pertenecen las imágenes o resultados de la función
  • Imagen: Valor que toma la función para cada x
  • Inyectividad, sobreyectividad y biyectividad

Construcción e interpretación de gráficos de funciones

Representación gráfica en el plano cartesiano. Análisis de la forma, crecimiento, decrecimiento y puntos críticos.

Aplicación del concepto de función a situaciones problemáticas

Modelización de fenómenos reales y ciencias aplicadas. Traducción de problemas a modelos matemáticos mediante funciones.

Modelización matemática de fenómenos

Tipos de funciones a analizar:

  • Funciones polinómicas: Expresiones con potencias enteras no negativas (ejemplo: f(x) = x³ – 4x + 2)
  • Funciones racionales: Cociente de dos polinomios (ejemplo: f(x) = (x² – 1)/(x + 2))
  • Funciones logarítmicas: Inversas de funciones exponenciales, con propiedades de crecimiento lento (ejemplo: f(x) = log(x))
  • Funciones exponenciales: Con base constante elevada a una variable (ejemplo: f(x) = 2ˣ)
  • Funciones trigonométricas: Seno, coseno, tangente, entre otras, con aplicación en fenómenos periódicos

3. Ecuaciones

Ecuaciones logarítmicas y exponenciales

Ecuaciones logarítmicas: Involucran logaritmos de variables, pueden transformarse aplicando propiedades logarítmicas para simplificar y resolver. Ejemplo: log(x) + log(x – 3) = 1

Ecuaciones exponenciales: Donde la variable está en el exponente; se solucionan aplicando logaritmos o igualando bases. Ejemplo: 2ˣ = 8 → x = 3

Σ Propiedad fundamental: logₐ(b) = c ⇔ aᶜ = b, y aˣ = aʸ ⇔ x = y (para a > 0, a ≠ 1)

Sistematización de algoritmos para la resolución de:

  • Ecuaciones logarítmicas: Aislar logaritmos, aplicar propiedades, pasar a forma exponencial, verificar dominio
  • Ecuaciones exponenciales: Intentar igualar bases, aplicar logaritmos si bases no son iguales, resolver ecuación resultante

Sistemas de ecuaciones

Pueden involucrar combinación de ecuaciones lineales, exponenciales o logarítmicas; se aplican métodos de sustitución, igualación o eliminación.

4. Límites y Continuidad

Conceptos y fundamentos

Concepto de límite: Valor al que se acerca f(x) cuando x se aproxima a un punto específico. Es fundamental para el cálculo diferencial.

Interpretación geométrica: Límite como el valor al que tiende la función en un punto, representado por la cercanía del gráfico a una altura (ordenada).

Concepto de continuidad: Función continua en x = a si existe f(a), existe el límite cuando x → a, y el valor del límite es igual a f(a).

Identificación de funciones continuas: Funciones polinómicas, exponenciales, logarítmicas y trigonométricas son continuas en sus dominios.

Σ Definición formal: limₓ→ₐ f(x) = L significa que para todo ε > 0 existe δ > 0 tal que si 0 < |x - a| < δ entonces |f(x) - L| < ε.

Cálculo e interpretación

  • Límites finitos: Valores concretos que puede tomar un límite; incluyen propiedades y resolución de indeterminaciones (ej. 0/0)
  • Límites infinitos: Cuando la función crece o decrece sin límites al acercarse a un punto o en el infinito
  • Límites exponenciales y trigonométricos: Se aplican fórmulas especiales y reglas para evaluar

Tipos de discontinuidades

  • Discontinuidad evitable: Cambio brusco que puede corregirse redefiniendo el valor en el punto
  • Discontinuidad esencial: Saltos, asíntotas o comportamientos no removibles

Reformulación de funciones: Redefinir o extender funciones para eliminar discontinuidades evitables.

5. Derivadas

Conceptos y fundamentos

Inferencia del concepto de derivada: Tasa de cambio instantáneo de una función respecto a una variable, límite del cociente incremental.

Interpretación geométrica: Pendiente de la recta tangente al gráfico en un punto dado.

Identificación de funciones derivables: Funciones continuas y suaves salvo excepciones en puntos de discontinuidad.

Σ Definición: f'(x) = limₕ→₀ [f(x+h) - f(x)]/h, cuando este límite existe.

Técnicas de cálculo

  • Derivada en un punto: Usando el límite del cociente incremental
  • Reglas de derivación:
    • Derivada de suma/diferencia: (f ± g)' = f' ± g'
    • Regla del producto: (fg)' = f'g + fg'
    • Regla del cociente: (f/g)' = (f'g - fg')/g²
    • Regla de la cadena: d/dx f(g(x)) = f'(g(x)) · g'(x)

Aplicaciones

  • Construcción e interpretación del gráfico de la derivada: Para determinar intervalos de crecimiento/decrecimiento y puntos críticos
  • Aplicación del concepto de derivada: Optimización, análisis de movimiento y tasa de cambio en fenómenos naturales y económicos
  • Modelización de situaciones: Uso de derivadas para describir dinámicas en física, biología, economía, etc.
  • Aplicación de algoritmos: Métodos numéricos para aproximar derivadas

Fórmulas Esenciales

Álgebra

Binomio al cuadrado: (a ± b)² = a² ± 2ab + b²

Binomio al cubo: (a ± b)³ = a³ ± 3a²b + 3ab² ± b³

Diferencia de cuadrados: a² - b² = (a + b)(a - b)

Cálculo

Reglas de derivación:

• (cf)' = cf'

• (f ± g)' = f' ± g'

• (fg)' = f'g + fg'

• (f/g)' = (f'g - fg')/g²

Problemas Resueltos

Ejemplo 1: Límite de función racional

Problema: Calcular limₓ→₂ (x² - 4)/(x - 2)

Solución: Factorizamos: (x² - 4) = (x - 2)(x + 2), entonces limₓ→₂ (x² - 4)/(x - 2) = limₓ→₂ (x + 2) = 4
