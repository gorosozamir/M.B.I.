Unidad 1: Revisión Gramatical Básica y Estructuras Fundamentales

Fundamentos gramaticales esenciales para la comunicación técnica en inglés.

Tiempos verbales simples

Present simple: Para hechos generales o verdades permanentes (e.g., "The processor runs at 3 GHz").

Past simple: Acciones terminadas en el pasado (e.g., "The system failed yesterday").

Future simple: Acciones que ocurrirán (e.g., "The update will install tomorrow").

There is / There are

  • Singular: "There is a file in the folder."
  • Plural: "There are many users connected."

Pronombres y adjetivos demostrativos

Este/esa/estos/esas: this, that, these, those. Ejemplo: "This device is faster than that one."

Caso posesivo

Formado con 's para singular y apóstrofe para plural (ej. "the computer's memory", "the users' data").

Adverbios de tiempo

Indican cuándo ocurre una acción: now, yesterday, today, tomorrow, always, sometimes.

Preposiciones comunes

In (en), on (sobre), at (en un lugar o tiempo específico), by (por, cerca de), among others. Ejemplo: "The file is saved in the folder."

Unidad 2: Tiempos Verbales Indefinidos y Aplicaciones Técnicas

Uso de tiempos verbales simples en contextos técnicos.

Present simple, Past simple, Future simple

Uso para describir rutinas, hechos pasados o planes futuros en informática.

"Used to" y "have to / had to"

"Used to" para hábitos pasados: "The company used to run older software."

"Have to / had to" para obligación: "You have to update the driver." / "They had to reboot the system."

Expresiones con "ago" y "how long ago"

Indican tiempo desde un evento: "The server crashed two days ago." / "How long ago did you install the update?"

Verbos regulares e irregulares

Ejemplos: "install" (regular: installed), "run" (irregular: ran), "come" (came).

Aplicación: describir materiales y procesos en computadoras

Ej: "Modern computers use silicon-based chips."

Unidad 3: Tiempos Verbales Continuos

Expresar acciones en curso en diferentes momentos temporales.

Present continuous / Past continuous / Future continuous

"The software is running." (Presente continuo)

"The system was updating when the power failed." (Pasado continuo)

"The technician will be testing the network tomorrow." (Futuro continuo)

Uso con vocabulario técnico

Descripción de procesos, actividades simultáneas o temporales en el funcionamiento de sistemas.

Unidad 4: Tiempos Verbales Perfectos

Acciones completadas con relevancia en otro momento temporal.

Present perfect

Para acciones pasadas con relevancia en el presente: "The update has been installed."

Past perfect

Para acciones que ocurrieron antes de otro evento pasado: "The server had restarted before the backup started."

Aplicación técnica

Seguimiento y registro de procesos completados sin especificar momento exacto.

Unidad 5: Condicionales y Consecuencias

Expresar condiciones y sus posibles resultados en contextos técnicos.

Condicionales tipo 0, 1 y 2

  • Tipo 0: hechos generales y verdades (If + present simple, present simple): "If the battery runs out, the device shuts down."
  • Tipo 1: posibilidades reales futuras (If + present simple, will + verbo): "If you update the software, it will improve performance."
  • Tipo 2: situaciones hipotéticas (If + past simple, would + verbo): "If I had more RAM, the program would run faster."

Estructuras modales

Obligatoriedad y posibilidad: "have to do," "had to do," "can," "must" (e.g., "You must reboot the router.").

Verbos defectivos

Verbos que carecen de ciertas formas, usados en condicionales y modales (e.g., must, can).

Unidad 6: Voz Activa y Pasiva

Estructuras para enfatizar diferentes elementos en la comunicación técnica.

Estructura pasiva en distintos tiempos

Present simple pasiva: "The software is installed by the technician."

Past simple pasiva: "The files were deleted accidentally."

Future simple pasiva: "The system will be upgraded next week."

Aplicación práctica

Describir procesos técnicos de forma impersonal (importante en manuales y documentación): "The antivirus is updated regularly."

Unidad 7: Cuantificadores y Expresiones de Cantidad

Expresar cantidades y cantidades en contextos técnicos.

some, any, a few, a little

  • "Some": en oraciones afirmativas: "There are some errors in the code."
  • "Any": en negativas e interrogativas: "Are there any bugs?" / "There aren't any viruses."
  • "A few": para plurales contables: "I have a few documents to review."
  • "A little": para incontables: "There is a little latency in the network."

How much? / How many?

Preguntas sobre cantidad: "How much memory does the device have?" / "How many users are online?"

Unidad 8: Adjetivos y Adverbios

Modificadores para descripciones técnicas precisas.

Uso y posición

Adjetivos preceden a los sustantivos: "A fast processor." Adverbios modifican verbos, adjetivos u otros adverbios: "The system runs quickly."

Comparativos y superlativos

Adjetivos cortos: "faster," "fastest."

Adjetivos largos: "more efficient," "most efficient."

Comparación de adverbios

"more carefully," "most carefully."

Aplicación práctica

Comparación de componentes y descripción técnica.
