Unidad 1: Estructuras Gramaticales

1.1 Futuro simple con "if" o "when" + presente simple

Uso: Se emplea para expresar consecuencias o condiciones futuras. La estructura es:

If/When + presente simple, will + verbo base.

Ejemplo:

  • If you connect all the computers, the system will run faster.
  • When the update finishes, the program will restart.

Notas:

  • "If" se usa para condiciones posibles.
  • "When" se utiliza cuando el resultado es seguro.

1.2 Future in the past: "was/were going to do"

Formación:

was/were + going to + verbo base

Uso:

Expresa planes o intenciones que existían en el pasado pero que no siempre se cumplieron.

Ejemplo: I was going to call you, but I forgot.

Aplicaciones específicas:

  • Tras expresiones como thought, never dreamt, had no idea:
    • I thought you were going to finish the report.
    • She never dreamt she was going to travel so far.
    • He had no idea they were going to cancel the meeting.
  • Tras told, said:
    • He told me he was going to update the software.
    • She said she was going to fix the bug.

1.3 Pasado perfecto (Past Perfect)

Formación:

had + participio pasado (past participle)

Uso:

Se utiliza para describir una acción que ocurrió antes de otra en el pasado.

Ejemplo: The system had crashed before the technician arrived.

Palabras asociadas:

  • just: She had just finished the installation when the error appeared.
  • ever: It was the first time he had ever configured a server.
  • never: They had never seen such a problem before.

1.4 Oraciones condicionales tipo 2

Estructura:

If + pasado simple, would/could + verbo base

Uso:

Se emplea para hablar de situaciones hipotéticas en el presente o futuro, poco probables o irreales.

Ejemplo: If the sensor weren't active, the system could fail.

Notas:

  • "Would" indica resultado probable; "could" posibilidad.

1.5 Voz pasiva (Passive Voice)

Formación:

be (en el tiempo correspondiente) + participio pasado

Aplicación en contextos técnicos:

Se usa para destacar la acción o el resultado más que al agente.

Ejemplo: New security systems are installed every year.

Notas:

  • El agente puede omitirse si no es relevante: The data was encrypted.

1.6 Lenguaje indirecto (Reported Speech)

Transformación:

Se transforma una oración o pregunta directa en indirecta, cambiando tiempos verbales, pronombres y expresiones temporales.

Ejemplo:

Directa: She said, "I use this software."

Indirecta: She said she used that software.

Cambios frecuentes:

  • Presente simple → pasado simple
  • Pasado simple → pasado perfecto
  • Futuro (will) → would
  • Pronombres y expresiones de tiempo/lugar cambian según contexto.

Unidad 2: Funciones Comunicativas y Contextos de Aplicación

2.1 Explicar beneficios del uso de computadoras en red

Ejemplo con futuro y condicional:

If computers are networked, sharing information will be easier.

Beneficios:

  • Facilita el intercambio de información
  • Optimiza recursos y procesos
  • Mejora la seguridad y el acceso a datos

2.2 Indicar la importancia de las computadoras en oficinas

Expresiones funcionales:

  • Computers are essential in modern offices.
  • They allow employees to manage documents, communicate, and analyze data efficiently.

Vocabulario relevante:

software, hardware, database, spreadsheet, network, printer, server, backup, user, administrator

2.3 Sistemas de almacenamiento de datos

Ejemplo con futuro:

When the system detects a failure, the backup will start.

Vocabulario técnico:

  • backup, storage device, cloud storage, data recovery
  • hard drive, redundancy, server, archive

Notas:

El uso de sistemas de almacenamiento modernos permite proteger la información y garantizar la continuidad operativa.

Ejercicios de estructuras gramaticales

Ejercicio 1: Completa con la forma correcta del verbo

  1. If you _____ (press) the button, the system _____ (start).
  2. When the installation _____ (complete), the computer _____ (restart) automatically.
  3. She told me she _____ (going to) update the software yesterday.
  4. The technician _____ (already/fix) the problem before we arrived.

Ejercicio 2: Transforma a voz pasiva

  1. They install new security systems every year.
  2. The technician repaired the server yesterday.
  3. Someone has encrypted all the data.
  4. They will test the new software next week.

Ejercicio 3: Transforma a lenguaje indirecto

  1. "I use this program every day," she said.
  2. "We have finished the installation," they told me.
  3. "I will send the report tomorrow," he said.
  4. "The system crashed yesterday," the technician explained.
